Hard and Soft Import Details

Field Name Description
Hard Import Hard Import will load the test cases in a way that the loaded test case acts as a different test case than the original test case; Any change made to the newly imported test case will not impact the original test case and vice versa. The two cases, imported and original, will act independently.
Soft Import Soft Import will load the test cases in a way that the loaded test case acts as the same test case as the original test case; Any change made to the newly imported test case will have a similar impact on the original test case and vice versa. The two cases, imported and original, will not act independently.
*More details can be found in go/uicd
